# Terima kasih telah menggunakan applikasi kami

Nahpu (NAtural History Project Utilities) adalah applikasi cross-platform untuk pencatatan dan pengelolaan data-data yang berkaitan dengan kegiatan penelitian di bidang biologi. Applikasi ini didesain untuk digunakan oleh para peneliti, mahasiswa, dan kurator di museum sejarah alam/biologi dan insitusi lain yang mengkoleksi dan mengelola sampel hayati.

Nahpu adalah applikasi gratis dan open source. Saat ini masih dalam tahap pengembangan. Versi saat ini mendukung pencatatan spesimen burung dan mamalia. Jika Anda ingin berkontribusi, silakan mengisi formulir di link ini: [https://forms.gle/6GPvmUaxsAqBSbmdA](https://forms.gle/6GPvmUaxsAqBSbmdA).
