# Bienvenido 🙏🏽🙏🏻🙏🏿

**_Repensando el Inventario de Biodiversidad en la Era Digital_**

NAHPU es una aplicación de catalogación de campo multiplataforma para la gestión de muestras biológicas. Está diseñado para ser utilizado por estudiantes y personal de la colección de historia natural y otras instituciones que recolectan y manejan especímenes biológicos.

NAHPU es un proyecto gratuito y de código abierto, y actualmente está disponible en [beta] (https://docs.NAHPU.app/en/installation). La versión actual admite especímenes de mamíferos y aves. Estamos trabajando en agregar herpetofauna.

## ¿Por qué NAHPU?

- **Diseñado para manejar trabajos de campo complejos**. Sin embargo, es amigable para principiantes.
- **Diseñado para el coleccionismo responsable**. Está desarrollado para maximizar la recopilación de datos en el campo, agilizar la gestión de datos y apoyar el trabajo de campo inclusivo.
- **Accesible para todos**. La documentación en varios idiomas está en proceso. También estamos trabajando para hacer que NAHPU sea accesible para personas con discapacidades.
- **Gratis y de código abierto**. NAHPU es de uso gratuito y está abierto a contribuciones.
- **Multiplataforma**. NAHPU está disponible para Windows, macOS, iOS, iPadOS y Android. Esperamos admitir Linux pronto.
