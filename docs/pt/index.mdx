# Bem-Vindo 🙏🏽🙏🏻🙏🏿

**_Repensando o Inventário da Biodiversidade na Era Digital_**

NAHPU é um aplicativo de catalogação de campo de plataforma cruzada para o gerenciamento de espécimes biológicos. Ele é projetado para ser usado por estudantes e equipes de coleta de história natural e outras instituições que coletam e gerenciam espécimes biológicos.

O NAHPU é um projeto gratuito e de código aberto e está atualmente disponível em [beta](https://docs.NAHPU.app/en/installation). A versão atual suporta espécimes de mamíferos e aves. Estamos trabalhando para agregar herpetofauna.

## Por que NAHPU?

- **Construído para lidar com trabalhos de campo complexos**. No entanto, é amigável para iniciantes.
- **Projetado para coleta responsável**. É desenvolvido para maximizar a coleta de dados no campo, simplificar o gerenciamento de dados de linha e apoiar o trabalho de campo inclusivo.
- **Acessível para todos**. A documentação em vários idiomas está em andamento. Também estamos trabalhando para tornar o NAHPU acessível para pessoas com deficiência.
- **Código aberto e gratuito**. O uso do NAHPU é gratuito e aberto a contribuições.
- **Plataforma cruzada**. O NAHPU está disponível para Windows, macOS, iOS, iPadOS e Android. Esperamos oferecer suporte ao Linux em breve.
